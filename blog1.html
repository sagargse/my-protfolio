1. What is Functions?

Functions as we know in every programming language this words occurs frequently.
A function is basically a block of code written to perform a specified actions. We have to define a fuction in our program to call it.

Defining a JavaScript function:-

  1. FUNCTION keyword
  2. FUNCTION Name
  3. A list of parameters seperated by commas inside the parenthesis ()
  4. Block of executable code inside the curly braces.

Syntax:-

function functionName()
{
//block of executable code
};

* FunctionName can contains letters,digits,underscore

Example :

//defining the function
function multiply(a,b){
return a*b;
};
//callling the function
multiply (4,5); //20

A return keyword ends the function execution and returns the specified value to the calling function from where it is called.

2. ARROW FUNCTIONS

1.Arrow functions were introduced in ES6.
2.It is the shorter version of the actual function syntax.

Let us take the above example of multiply fuction through arrow function

var multiply = (a,b) => {
return a*b;
};
Console.log(multiply(2,3)); //6
