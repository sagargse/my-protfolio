|| Introduction to JSON ||

> JSON stands for JavaScript Object Notation.

> JSON is commonly used for transporting data.

> As we have created JavaScript object the JSON format is almost similar to the JavaScript objects.

> JSON is a plain-text format.

|| SYNTAX ||

Rules:

  1. Data should be key/value pairs.
  2. Commas seperates the data.
  3. Object should be in curly braces.
  4. Arrays are holds in Square brackets.

* key requires to be in double quotes.
* then followed by colon.
* then value.

JSON Data example:

"name" : "sagar"

* In JSON keys must be strings while in JavaScript keys can be strings,numbers or identifier names.

JSON object {"name" : "sagar"}

JavaScript object {name : "sagar"}

In JSON values must be of the following types

  1. string
  2. Number
  3. Object
  4. Array
  5. Boolean
  6. Null

* While in JavaScript the value can be of all type includings function , data , expression.
